Welcome to the sites page for the ldap scrabble badge. This badge is awarded to users based on how many points their LDAPs would get in English editions of Scrabble. The point values for each letter can be found here: http://en.wikipedia.org/wiki/Scrabble_letter_distributions#English. Note: The script I had running weekly to update this badge stopped working, and I haven't gotten around to fixing it.
